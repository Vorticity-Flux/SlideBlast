What is SlideBlast?

SlideBlast is a realtime, web-based presentation tool. It lets you display slides to multiple people through the web, and ensures that all attendees are viewing the same slide.

Does SlideBlast provide voice conferencing?

Nope. You will need to arrange for your own voice communication. We suggest telephones.

What file types are allowed in a .zip file?

Your .zip file can contain the following file types:
  • Images - .gif, .png, .jpg
  • Plain Text - .txt .markdown, .xml
  • Code - .erl, .cs, .cpp, .js, .java, .sh, .sql,
